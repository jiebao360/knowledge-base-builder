## 📋 技能描述

帮助 OpenClaw 用户快速搭建个人知识库系统，包含飞书知识库空间创建、文档导入、分类配置、自动收录等完整流程。

**核心功能**:
- ✅ 创建飞书知识库空间
- ✅ 导入 12 个技能模块文档
- ✅ 配置 Content Collector 自动收录
- ✅ 设置 8 大分类体系
- ✅ 建立索引和搜索

**使用技能**:
- `feishu-create-doc` - 文档创建
- `feishu-wiki-space` - 知识库管理
- `elite-longterm-memory` - 记忆管理

## 🎯 适用场景

- ✅ 想建立系统化知识管理体系

## 🚀 快速开始

### 1. 安装技能

```bash
# 从 Clawhub 安装（推荐）
安装 https://clawhub.ai/你的用户名/knowledge-base-builder

# 或从 GitHub 安装
npx skills add 你的用户名/knowledge-base-builder -g -y
```

### 2. 运行技能

在 OpenClaw 对话中发送：
```
帮我搭建个人知识库
```

或带参数运行：
```
帮我搭建个人知识库，使用飞书知识库，配置 8 大分类
```

### 3. 跟随指引

技能会引导你完成：
1. 飞书授权确认
2. 知识库空间创建
3. 文档模板选择
4. 分类体系配置
5. 自动收录设置

## 📊 执行流程

### Phase 1: 准备工作 (2 分钟)
```
1. 检查飞书授权状态
2. 检查必需技能是否安装
3. 确认用户偏好（本地/飞书）
```

### Phase 2: 创建知识库空间 (3 分钟)
```
1. 创建飞书知识库空间
2. 设置空间名称和描述
3. 配置访问权限
4. 创建 6 个子目录
```

### Phase 3: 导入文档 (5 分钟)
```
1. 创建知识库首页 (README)
2. 导入 12 个技能模块文档
3. 创建项目案例文档
4. 创建 SOP 文档
5. 创建资源配置文档
6. 创建索引文档
```

### Phase 4: 配置分类 (2 分钟)
```
1. 设置 8 大分类标签
2. 配置自动归类规则
3. 建立关键词索引
```

### Phase 5: 配置自动收录 (3 分钟)
```
1. 安装 Content Collector (如未安装)
2. 配置收录规则
3. 设置分类映射
4. 测试自动收录
```

### Phase 6: 完成验收 (2 分钟)
```
1. 检查所有文档是否创建
2. 验证分类体系
3. 测试搜索功能
4. 生成使用报告
```

**总预计时间**: 17 分钟

## 🔧 配置选项

### 基础配置
```json
{
  "storage": "feishu",  // 或 "local"
  "space_name": "OpenClaw 个人知识库",
  "categories": 8,  // 分类数量
  "auto_sync": true  // 自动同步到飞书
}
```

### Content Collector 配置
```json
{
  "enabled": true,
  "categories": {
    "技术教程": ["教程", "指南", "文档"],
    "实战案例": ["案例", "项目", "实战"],
    "产品文档": ["SOP", "流程", "规范"],
    "学习笔记": ["笔记", "心得", "总结"],
    "热点资讯": ["资讯", "新闻", "动态"],
    "设计技能": ["设计", "UI", "视觉"],
    "工具推荐": ["工具", "资源", "API"],
    "训练营": ["课程", "培训", "学习"]
  }
}
```

## ⚠️ 注意事项

### 飞书权限
- 需要 `bitable:app` 权限
- 需要 `docs:doc` 权限
- 需要 `drive:file` 权限
- 需要 `wiki:space` 权限

### 文件大小
- 单个文档最大 10MB
- 知识库总空间 1GB
- 图片自动压缩

### 同步频率
- 手动同步：随时
- 自动同步：每日 22:00
- 变更同步：实时（可选）

## 🐛 常见问题

### Q1: 飞书授权失败怎么办？
**A**: 在飞书中找到 OpenClaw 机器人，重新发送授权请求，确认所有权限。

### Q2: 文档创建失败？
**A**: 检查飞书 API 连接，或改用本地模式 (`storage: "local"`)。

### Q3: Content Collector 不工作？
**A**: 先安装 Content Collector 技能，然后配置收录规则。

### Q4: 可以自定义分类吗？
**A**: 可以！修改配置中的 `categories` 字段。

### Q5: 能导出到其他平台吗？
**A**: 支持导出为 Markdown，可导入 Notion、Obsidian 等。
